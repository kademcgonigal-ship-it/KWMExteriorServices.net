# KWM Exterior Services

A marketing website for KWM Exterior Services — a lawn care, landscaping, and exterior maintenance business. Built with TanStack Start and deployed on Netlify.

## Tech Stack

- **Framework:** TanStack Start (React 19 + TanStack Router v1)
- **Build:** Vite 7
- **Styling:** Tailwind CSS 4 + custom CSS (BEM-style `kwm-*` classes)
- **Language:** TypeScript 5.7 (strict mode)
- **Deployment:** Netlify

## Features

- Sticky header with smooth anchor navigation
- Full-viewport hero section with call-to-action buttons (call, text, email)
- Services grid showcasing six service offerings
- About section with imagery
- Customer reviews section
- Contact footer with free estimate offer

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts at [http://localhost:3000](http://localhost:3000). Use the Netlify CLI for full platform feature emulation:

```bash
netlify dev
```

This runs on port 8888 and emulates Netlify Functions, redirects, and other platform features.
